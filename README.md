# Car Rental Application (ASP.NET Core Web API)

This is the **backend API** for my **Car Rental Application**, built with **C#** and **ASP.NET Core Web API**.  
It is designed to work with an **Angular frontend**, handling CRUD operations, authentication, bookings, and favorites.

## Frontend Skills
- Angular, TypeScript, HTML5, CSS3, Angular Material, SweetAlert2  
- Responsive UI, Reactive Forms, Observables, Routing, Reusable Components, State Management

## Backend Skills
- C#, ASP.NET Core Web API, Entity Framework Core, LINQ  
- SQL Server, Migrations, Repository Pattern, AutoMapper, DTOs, Dependency Injection  
- RESTful APIs, Validation, Error Handling, Security Best Practices

## Features
- Full CRUD operations for Cars, Users, and Bookings  
- Favorites system and Booking workflow  
- Secure API endpoints with authentication & authorization  
- Clean integration with Angular frontend

## How to Run
1. Clone the repo  
2. Open in Visual Studio  
3. Update `appsettings.json` with your SQL Server connection  
4. Run `dotnet ef database update` to apply migrations  
5. Run the API and test endpoints with Postman or integrate with Angular frontend
